Description:
Abstract:
The Louisiana State Legislature created Coastal Wetlands Planning, Protection, and Restoration Act (CWPPRA) in order to conserve, restore, create and enhance Louisiana's coastal wetlands. The restoration plans developed persuant to these acts specifically require an evaluation of the effectiveness of each coastal wetlands restoration project in achieving long-term solutions to arresting coastal wetland loss. This data set consists of digital data describing the land/water classification for the Highway 384 Hydrologic Restoration CS-21 for the year 2002. This data is used by project managers at each site in order to get a general overview of land loss. This data can be compared to the same type of data from previous years to conduct trend analysis.
Purpose:
The intended use of this data set is to provide information to aid efforts in the conservation, restoration, creation and enhancement of Louisiana's coastal wetlands in order to aid in long-term solutions to these problems.

Using the Imagine software, two separate classifications are performed on the same image. 100 random pixels are tested from the two classifications. If the classifications do not match at least 80%, the values are recalculated and the test reperformed.

Process_Step:
Process_Description:
Primary Classification: An unsupervised classification is performed using Imagine software on the previously prepared photo mosaic. The classification program groups the pixels into 50 different classes and 10 iterations based upon a range of colors. These classes are then determined to be either land or water- consultation with the original aerial photography may be neccesary if a classification is in question. The finished classification is recoded into the two classes; occasionally a third class, mudflat, can be utilized.

Entity_and_Attribute_Overview:
Each pixel within the project boundary is assigned a classification (50 classes, 10 iterations) based on the pixel's color. An unsupervised classification has a preset range of colors to determine class. The 50 classes are then grouped into two classes. land and water.
