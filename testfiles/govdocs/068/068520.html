Fixed p17 Data, p17.09.00 Data And Monte Carlo

Please switch to the new data sets for summer.
Here you can find information on how to access
  • unskimmed data
  • skimmed thumbnails (including p17.09.00 data)
  • skimmed CAF trees and
  • Monte Carlo CAF trees.

Unskimmed Data

All data reconstructed with p17.03.03 and p17.05.01 needs fixing. You can use the following SAM constraints to find unskimmed fixed data (warning: this takes a long time): 

sam translate constraints --dim="(((APPL_NAME  p17tmbfixer and (VERSION  p17.07.00 or VERSION p17.07.01)) and DATA_TIER  thumbnail) and FILE_NAME  TMBfix-recoT_all_%)"

All data reconstructed with p17.09.00 already includes the fixing. You can find the unskimmed data with these constraints: 

sam translate constraints --dim="(((APPL_NAME  recon_root and VERSION  p17.09.00) and DATA_TIER  thumbnail) and FILE_NAME  recoT_all_%)"

To learn how to process inclusive skims efficiently, see this HOWTO on filtering based on event flags.

Skimmed Thumbnails

Please use the following predefined dataset definitions in SAM to find skimmed thumbnails

  • CSskim-SKIM-PASS2-p17fix (Fixed data)
  • CSskim-SKIM-PASS2-p17.09.01 (p17.09 data)

or use a command like this from the shell: 

  • sam translate constraints --dims="(((APPL_NAME tmbskim and VERSION csskim-pass2-p17) and DATA_TIER thumbnail) and FILE_NAME CSskim-XXXXXX-%)"
    (Fixed data)
  • sam translate constraints --dims="(((APPL_NAME tmbskim and VERSION csskim-pass2-p17.09.01) and DATA_TIER thumbnail) and FILE_NAME CSskim-XXXXXX-%)"
    (p17.09 data)
where XXXXX stands for the skim name (see below).

Skimmed CAF Trees

To find CAF trees use one of the predefined dataset definitions

  • CSG_CAF_SKIM_v3
  • CSG_CAF_SKIM_v3_p17.09.01 for p17.09 data
or a command like this: 
sam translate constraints --dims="(((DATA_TIER  root-tree-bygroup and APPL_NAME  tmb_analyze) and VERSION  csgcaf_01.01.00) and FILE_NAME  CAF-CSGv1-CSskim-XXXXXX-%)"
where again XXXXXX stands for the skim name. For p17.09 data replace the version with csgcaf_01.03.00

Smeared Monte Carlo

To find CAF trees for Monte Carlo, use the dataset definition
  • CSG_CAF-MCv2-XXXXX
or the following sam command 
sam translate constraints --dims="((((RUN_TYPE 'monte carlo') and DATA_TIER  root-tree-bygroup and APPL_NAME  tmb_analyze) and VERSION  csgcaf_01.02.00) and GLOBAL.REQUESTID XXXXX)"
where XXXXX is the official Monte Carlo request ID.

These CAF trees are produced with p18.04.00.

Unsmeared Monte Carlo

If you really want the older unsmeared Monte Carlo (where trees are not available for all requests) use this constraints: 
sam translate constraints --dims="((((RUN_TYPE 'monte carlo') and DATA_TIER  root-tree-bygroup and APPL_NAME  tmb_analyze) and VERSION  csgcaf_01.01.00) and FILE_NAME CAF-MCv1-XXXXX-%)"
or the dataset definitions:
  • CSG_CAF-MCv1-XXXXX

These CAF trees are produced with p18.03.00.
