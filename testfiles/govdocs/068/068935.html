SEARCH NOTES FOR CLASS D11
(1) Include vinaigrette, or apparel-attached holder for flower, fragrance, etc.
(2) Includes watch strap. For strap in combination with watch, see D10-32.
(3) For charm or pendant per se, see subclasses 79+.
(4) For individual link, fastener, etc., see subclasses 86+.
(5) For watch fob or chain, see subclasses 79+.
(6) Classified by major design elements; small connector elements are disregarded.
(7) For individual link, see subclasses 86+.
(8) For setting, see subclasses 91+.
(9) For stone or gem, see subclasses 89+.
(10) For badge or medal, see subclasses 95+.
(11) For brooch containing claimed letter or numeral, see subclass 61.
(12) Patents are placed in this subclass and indented subclasses only if they disclose realistic representations of, and are positively identifiable as, plant life forms. For boutonniere, corsage, etc., see subclasses 117+.
(13) Patent is placed in this subclass only if an element is shown to be passing entirely under, or weaving through, itself or another element.
(14) For crucifix, see D99-27. For medal, see subclasses 95+.
(15) Includes automobile badge, insignia, etc; includes coin or token; includes military or fraternal order insignia. For emblem per se, see subclass 99.
(16) For light string, light reflector or candle holder, see D26, subclasses 25, 128+, and 9+, respectively. For religious appliance, see D99-25. Includes tinsel.
(17) For statue, see subclasses 143+.
(18) Includes skirt for Christmas tree base.
(19) For vase or flower holder, see subclasses 143+; includes party favor; includes trophy.
(20) For picture frame, see D6-300+.
(21) For wall-mounted plant support, see D6-553+; includes vase; includes combination with drain saucer.
(22) For aquarium, see D30-6+.
(23) Includes flagpole or banner rod.
(24) Includes inedible cake or pastry decoration; Includes emblem; Includes package ornament or bow.
(25) Subject matter in this subclass is distinguished from subject matter in D2-624+ in being fasteners for apparel whereas the latter includes only articles which attach apparel to the body. For example, belts, suspenders and garters are classified in D2. Classified by full line diclosure.
(26) Includes hankerchief ring. For tie pin, see subclass 222.
(27) For pin which is intended for adornment only, and not for fastening, i.e., stickpin, see subclass 47.
(28) For stud per se, see subclass 222.
(29) For buckle with prong engagement, see subclass 231.
(30) Includes letter or numeral.
(31) For sets having mirror image symmetry, see subclass 213.
(32) Exclusive of mounting hole.
(33) Includes humanoid.
(34) Includes letter numeral.
(35) For fastening pin, e.g., hatpin type, see subclass 207.
(36) For potpourri pot, see D23-366+.
