System Messages
Last Updated: 06/30/2008 @ 13:17
Message: On June 24, 2008 a new version of the Audit/NCR System was put into production. This new web site is located at "https://auditncr.gsfc.nasa.gov".

There may be a few open audits and NCR records that are still open at this time. These audits and NCRs will continue to be processed in the old applications until they are closed.

User IDs have been transferred to the new Audit/NCR application. For access to the new application, your user ID will need to be reset. To request a reset of your user ID, use the "Contact Administrator" link at the top of the new application.

Thank you.
