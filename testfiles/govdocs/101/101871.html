Search EMAP Bibliographic Database

This screen is designed to access the EMAP bibliography. The entire set of citations can be retrieved by pressing the submit button. Searches can be focused by specifying values for one or more of the fields below. Fields with lists of items will accept multiple selections by holding down the Ctrl key. When multiple items are selected for a field i.e., Authors, records that match any, not all, of the selections will be returned. When items are selected from multiple fields i.e., Authors and Keywords, records that match all of the selections will be returned. To start a new query, press Ctrl R or the Reset button to refresh the screen.
