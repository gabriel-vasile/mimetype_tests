You can run the tutorials locally from your computer without the need for an Internet connection. Each tutorial is made up of about 800 files.

  • Save the zipped file to your computer
  • Open the zipped file (double click on filename, e.g. Tox1.zip)
  • Double click on the index.htm file to open and run the tutorial.

You will need the free utility WINZIP® to open the zipped file.
