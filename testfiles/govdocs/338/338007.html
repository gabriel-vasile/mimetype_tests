Allowing Pop-Ups
Last Updated: December 21, 2007

The presence of an active "pop-up" blocker may interfere with the use of the Court's payment window in CM/ECF. Depending on the type of pop-up blocker present, the user may have to either modify the pop-up blocker to allow pop-ups from the Court or disable the pop-up blocker.

  • Internet Explorer SP2 7.0

  • Mozilla Firefox

  • Mozilla

  • Google Toolbar

  • Yahoo Toolbar

  • Spyware Doctor

  • Stop-Sign

INTERNET EXPLORER SP2 7.0

  1. Open Internet Explorer.

  2. Select Tools from the menu bar.

  3. Select Pop-up Blocker.

  4. Select Pop-up Blocker Settings.

  5. Type uscourts.gov in the "Address of the Web site to allow" field and click the Add button.

    Note: A temporary solution is to disable the Pop-up Blocker by selecting the Turn Off Pop-up Blocker command from Internet Explorer's Tools | Pop-up Blocker menu.

MOZILLA FIREFOX (v2.0 and above)

  1. Open Firefox.

  2. Select Tools from the menu bar.

  3. Select Options from the drop down menu.

  4. Select the Content tab.

  5. Select Exceptions -across from Block pop-up windows.

  6. Type uscourts.gov in the "Address of web site" field and click the Allow button.

  7. Click close button.

  8. Click OK button to complete the process and return to your browser.

MOZILLA

  1. Open Mozilla.

  2. Select Edit from the menu bar.

  3. Select Preferences from the drop down menu.

  4. Expand the Privacy & Security menu by clicking on the plus ( + ) next to words Privacy & Security.

  5. Click on Pop-up Windows under the Privacy & Security menu.

  6. Select Allowed Sites.

  7. Type uscourts.gov in the "Allow popups from the following web sites" field and click the Add button.

  8. Click OK button.

  9. Click OK button to complete the process and return to your browser.

GOOGLE TOOLBAR

  1. Open your browser.

  2. Select Options from the Google toolbar.

  3. In Accessories, uncheck Popup Blocker.

  4. Click OK button to complete the process and return to your browser.

YAHOO TOOLBAR

  1. Open your browser.

  2. Click on the Pop-Up Blocker icon from the Yahoo toolbar.

  3. Select "Always Allow Pop-Ups From..." from the drop down menu.

  4. Highlight the site name [uscourts.gov] in the "Sources of Recently Blocked Pop-Ups" field and click the Allow button.

  5. Click OK button.

  6. Click Close button to complete the process and return to your browser.

SPYWARE DOCTOR (Currently, Internet Explorer is the only browser integrated with Popup Blocker.)

  1. To temporarily disable Popup Blocker, Hold the [CTRL] key down while clicking on the URL that generates the popup.

STOP-SIGN

  1. From the Stop-Sign Popup Blocker Settings dialog window, click the Edit List button in the Blocked popups section.

  2. Type the URL into the Add this webiste/URL to the list field.
    OR
    Copy the URL from the address bar of your browser and paste it into the Add this website/URL to the list field.

  3. Click the Add button (enabled only after you have added text).

  4. The URL appears in the list.

  5. Click OK to close the window.
