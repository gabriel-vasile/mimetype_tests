Pop Up Setting

 

Instructions

 

A pop-up blocker in Internet Explorer was installed with Windows Service Pack 2. Which may block some features of Transcom.

Pop-up Blocker is turned on in Internet Explorer and set to the medium setting by default, which means it will block most automatic pop-ups. The default settings for the pop-up blocker allows you to see pop-ups that are opened when you click a link or button on a Web site. Pop-up Blocker will also play a sound and show the Information Bar when a pop-up is blocked. You can adjust these settings so that Pop-up Blocker works the way you want it to.

To allow all TRANSCOM alerts and features to work correctly you must add http://transcom.energy.gov/app/start domain to your pop-up blocker settings. To add this domain follow the directions below:

Setting Pop-up Blocker

1.

Open Internet Explorer.

2.

On the Tools menu, point to Pop-up Blocker, and then click Pop-up Blocker Settings.

3. Type in http://transcom.energy.gov/app/start in the "Address of Web site to allow" bar.

4. Click ADD and then click Close on the bottom.

5. Pop-up windows should now work correctly for the Transcom website and TRANSCOM monitoring system.

** If you have recently installed Internet Explore 7 you will have to re-add http://transcom.energy.gov/app/start to the pop up blocker then close and reopen your browser to have changes take effect.**
