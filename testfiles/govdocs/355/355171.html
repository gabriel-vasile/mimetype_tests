ARMoS: A Relational Modeling Scheme


Work by
Alper Atamturk
Ellis Johnson
Jeff Linderoth
Martin Savelsbergh

We have built a system (called ARMoS) based on relational database technology in which all modeling related activites -- formulation, data instantiation, and solution manipulation are done using SQL. This modeling scheme offers an alternative to (and some advantages over) modeling languages such as GAMS, AIMMS, and AMPL. A report on this work has been published under Report LEC-96-10.
