1 //- Class: GetLongOpt
2 //- Description: GetLongOpt manages the definition and parsing of
3 //- long options. Long options can be abbreviated as long as there
4 //- is no ambiguity. If an option requires a value, the value should
5 //- be separated from the option either by whitespace or an "=".
6 //-
7 //- Other features: $
8 //- o   GetLongOpt can be used to parse options given through environments.$
9 //- o   GetLongOpt provides a usage function to print usage.$
10 //- o   Flags & options with optional or mandatory values are supported.$
11 //- o   The option marker ('-' in Unix) can be customized.$
12 //- o   Parsing of command line returns optind (see getopt(3)).$
13 //- o   Descriptive error messages.$
14 //-
