Data Element Explanation

 

Element Label: DMU Description

Description:

A brief description that identifies the meaning of the data mapunit for future reference. The muid was entered in this column during conversion from SSSD because the muid provided a connotative reference to SSSD data. As new data mapunits are created, a reference to the major named components and phase characteristics, such as "Woodbridge stony, 0 to 3" may be more appropriate.

 

Specific data mapunits related to a legend mapunit can be found by using a query or the Find Related function.
