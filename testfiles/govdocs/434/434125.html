NASIS Ecosite Component Text Note Guide

Updated Apr 17, 2008

The following document requires Adobe Acrobat:

Printable Version: Ecosite Component Text Note Guide (PDF; 46 KB) 

If the keyed Ecosite is different from the assigned/correlated Ecosite, a component text note should be created.

The following is an example of the procedure to follow in NASIS. In this example, a component text note is created for a component in MLRA 43B, LRU A with a keyed ESD of Sandy Argillic that has been correlated to a Sandy ESD in MLRA 43B and LRU A.

  1. In NASIS, select View → Components → Component Text
  1. In the component text table, press “F8” to insert a new row.
  2. In the new row, enter Date – “ / / ”; Author – “your name” (no initials); Kind – “miscellaneous notes”; Category – “Keyed ESD”; Subcategory – Keyed Ecological Site ID, ex. “R044BA115MT”; and Text – “Keyed: SyA, Ver. 7.2”.

Note: Ver. 7.2 refers to the Ecosite Key Version. Enter the appropriate version.

The Ecosite code is connotative and reflects MLRA, LRU, Ecosite, and State. Below is an example of an Ecosite ID.

R044BA115
