Goddard Space Flight Center
2001 Phase 2

A Distributed Cluster Based Architecture for Constellations

WW Technology Group

Ellicott City, MD

INNOVATION

Software based infrastructure that facilitates dependable formation and operation of distributed real-time control systems. Several innovations advance the state of the art in distributed systems technology, including: (1) cold start algorithms for autonomous, dependable system organization and formation, (2) distributed on-line configuration control, monitoring and health assessment processes operating in concert across the entire system for coordinated system operation, (3) a multi-level scheduling approach, so distributed control elements will be responsive to local events and participate in system wide coordinated operations and (4) a robust fault/error model accounting for arbitrary fault/error cases and anomalies, thereby providing superior fault tolerance, particularly for Single Event Upsets.



ACCOMPLISHMENTS

  • Establishment of a System Organization Framework, initially populated with a Distributed Fault Tolerant Element Discovery Protocol and a local start-up sequencer
  • Production of a Baseline Health Monitoring service that provides coverage for the fault/error model
  • Production of an interface library through which hosted applications may obtain dependability services
  • Implementation of a 4-node prototype
COMMERCIALIZATION
  • Distribution of dependability middleware provided to the Formation Flying Test Bed group for evaluation
  • Key dependability technology on the winning Honeywell ST-8 program for Environmentally Adaptive Fault Tolerant Computing
  • Flexible and extendable with multiple applications
  • Critical roadmap step for distributed space systems, including Spacecraft Constellations and Sensor Webs, serving as a basis for Dependable Virtual Platforms
 GOVERNMENT/SCIENCE APPLICATIONS
  • Critical Enabling Technology for Distributed Spacecraft and key for dynamic formulation of Sensor Webs:
    • Infrastructure is key to discovering available nodes and initializing a group of distributed spacecraft or forming a sensor web
    • Enables sensor webs to discover elements and form web autonomously
  • Fits GMSEC architecture and middleware-oriented paradigms
